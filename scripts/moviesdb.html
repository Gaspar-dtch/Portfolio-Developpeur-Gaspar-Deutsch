MOVIES DB

HTML, CSS, JS, API
Site
Github Code

Lors du TP final du module JavaScript, j'ai réalisé un site web basé sur l'utilisation d'une API de films, permettant d'afficher et de gérer un catalogue et un affiche des données de manière dynamique. La page d'accueil présente les films sous forme de catalogue, avec un chargement des données depuis l'API. En cliquant sur un film, l'utilisateur peut accéder à plusieurs informations comme le réalisateur, l'année de sortie ou les catégories associées. Le site propose différentes fonctionnalités côté interface, comme un système de like/dislike, ainsi que la possibilité d'ajouter, modifier et supprimer des films directement depuis le site. Ces actions influencent l'affichage du catalogue et les données manipulées tout au long de la navigation. Une attention particulière a été portée à la barre de recherche et au système de filtres par catégories, qui fonctionnent ensemble pour affiner les résultats affichés. Cette partie a été l'une des plus complexes à mettre en place, car elle demandait de bien gérer la logique JavaScript et les données récupérées. Ce projet m'a permis de renforcer ma compréhension de l'utilisation d'une API en JavaScript, ainsi que la gestion de données dynamiques et d'interactions utilisateur sur un site web.
